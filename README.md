# smart-calorie-calculator-and-fitness-web-application-
A full-stack fitness web application that calculates BMI, BMR, daily calorie needs, and macronutrient distribution, while also providing diet suggestions, visual insights, and an interactive chatbot.
it is a research based project
🚀 Features

• Calculates BMI, BMR, and daily calorie requirements
• Provides goal-based diet plans (lose, maintain, gain)
• Displays macronutrient breakdown (protein, carbs, fats)
• Generates visual charts and fitness reports
• Includes a hybrid chatbot system:

Rule-based keyword matching using a custom dataset
OpenAI API fallback for complex or unmatched queries
🧠 How it Works

User inputs are processed through a PHP backend where calculations are performed. The chatbot first checks user queries against predefined keywords and dataset responses. If no match is found, it intelligently switches to the OpenAI API to generate accurate responses.

🛠️ Tech Stack

HTML • CSS • JavaScript • PHP • Chart.js • OpenAI API
